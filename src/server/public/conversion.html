About Currency Conversion

This module automatically converts transactions in YNAB accounts where transactions are imported in a different currency than the budget currency.

For example: if your budget is in USD but bank imports transactions in EUR, this module will convert them to USD using exchange rates from the database.

The original amount is preserved in the memo field (e.g., "123.45 EUR | Original memo").
